## 🔐 アイデンティティと信頼

エージェントが「何者か」を証明する。

- [Chitin](https://chitin.id/) - ERC-8004 ソウルバウンドトークンによるAIエージェントのオンチェーン出生証明書。Base L2上で動作。
- [World ID](https://world.org/) - 人間とAIエージェントを区別する人格証明プロトコル。
- [Trusta.AI](https://trustalabs.ai/) - 5次元のSIGMA信頼モデルでエージェントをスコアリングするユニバーサル信用レイヤー。
- [Agentscan](https://agentscan.info/) - ZK-Proofレピュテーション検証付きのERC-8004エージェント検索エンジン。
- [Beyond Identity AI Trust Layer](https://www.beyondidentity.com/products/ai-trust-layer) - デバイスバウンドIDとMCPサーバー制御によるエンタープライズ向けエージェントセキュリティ。

## 💰 決済と清算

エージェントがどう支払い、どう稼ぐか。

- [x402 Protocol](https://www.coinbase.com/developer-platform/products/x402) - HTTPネイティブの決済プロトコル。Base上のUSDCで402ステータスコードを使用。Coinbase製。[GitHub](https://github.com/coinbase/x402)
- [Google AP2](https://ap2-protocol.org/) - 暗号化「マンデート」による検証可能な意思表示を持つエージェント決済標準。Salesforce、Visa、Mastercard参画。[GitHub](https://github.com/google-agentic-commerce/AP2)
- [Visa TAP](https://developer.visa.com/capabilities/trusted-agent-protocol) - チェックアウト時に正当なAIエージェントとボットを区別するオープンフレームワーク。[GitHub](https://github.com/visa/trusted-agent-protocol)
- [Skyfire](https://skyfire.xyz/) - USDCウォレット、支出制限、使い捨てクレデンシャルを備えたAIエージェント向け決済ネットワーク。
- [Nevermined](https://nevermined.ai/) - エージェント間コマースのための分散型決済レール。MCP、A2A、x402をサポート。
- [Coinbase AgentKit](https://www.coinbase.com/developer-platform/products/agentkit) - ガスレスSmart Walletトランザクション付きのAIエージェント向け暗号ウォレットツールキット。[GitHub](https://github.com/coinbase/agentkit)
- [Fetch.ai](https://fetch.ai/) - 世界初のAI間実取引決済を実現した自律エージェントフレームワーク。

## 🤝 交渉とコマース

エージェントが取引する。

- [Haggle Protocol](https://haggle.dev/) - トラストレスエスクローとアルゴリズムによる価格発見を備えた、AIエージェント初のオンチェーン交渉プロトコル。
- [UCP](https://ucp.dev/) - AIエージェントが商品を発見・購入するためのユニバーサルコマースプロトコル。GoogleとShopifyによる共同開発。
- [ACP](https://www.agenticcommerce.dev/) - OpenAIとStripeによるエージェンティックコマースプロトコル。エージェント駆動コマースのオープン仕様。[GitHub](https://github.com/agentic-commerce-protocol/agentic-commerce-protocol)
- [NEAR AI](https://near.ai/) - AIエージェントがタスクに入札し、NEARトークンで決済を受ける分散型マーケットプレイス。

## 📰 出版とコンテンツ

エージェントがコンテンツを作り、配信する。

- [Hum](https://hum.pub/) - skill.md統合、Trust Score、ERC-8004証明書、収益分配を備えたAI著者出版プラットフォーム。
- [Moltbook](https://www.moltbook.com/) - Reddit風コミュニティ（Submolts）を持つAI専用ソーシャルネットワーク。150万以上のAIエージェントユーザー。
- [Virtuals Protocol](https://www.virtuals.io/) - BaseとSolana上でコンテンツを生成・収益化するトークン化AIエージェントのローンチパッド。

## 🌐 プラットフォームとソーシャル

エージェントが暮らし、交流する場所。

- [MoChat](https://mochat.io/) - AIエージェントが独自のアイデンティティと認証を持つファーストクラス市民として扱われるエージェントネイティブメッセージングプラットフォーム。[GitHub](https://github.com/HKUDS/MoChat)
- [ClawdChat](https://clawdchat.ai/) - nanobotやPicoClawとskill.mdで統合されたエージェントソーシャルネットワーク。
- [Moltter](https://moltter.net/) - AIエージェント向けソーシャルネットワーク。エージェントが「molt」を投稿し、フォローし合い、エンゲージメントを構築。
- [SingularityNET](https://singularitynet.io/) - 分散型AIマーケットプレイス。ASI Alliance（Fetch.ai、Ocean Protocolと統合）の一部。
- [Morpheus](https://mor.org/) - LLMをウォレット、dApps、スマートコントラクトに接続するP2Pパーソナルエージェントネットワーク。[GitHub](https://github.com/MorpheusAIs/Morpheus)

## ⭐ レピュテーションとスコアリング

時間をかけて信頼を構築する。

- [ERC-8004 Reputation Registry](https://eips.ethereum.org/EIPS/eip-8004) - Ethereum Attestation Serviceを通じた構造化・検証可能なエージェントフィードバックのオンチェーンレジストリ。
- [Ethereum Attestation Service](https://attest.org/) - オンチェーン・オフチェーンアテステーションのオープンプロトコル。ERC-8004レピュテーションの基盤。[GitHub](https://github.com/ethereum-attestation-service/eas-contracts)
- [8004scan](https://www.8004scan.io/) - 「AIエージェント版Etherscan」— ERC-8004エージェント、レピュテーションスコア、リーダーボードを閲覧。AltLayer製。
- [TrustGo](https://trustalabs.ai/trustgo) - 人間向けMEDIAスコアからAIエージェント向けSIGMAへ移行中。CelestiaとStarknetで採用。Trusta.AI製。

## 📐 標準とプロトコル

すべてを支える基盤技術。

- [ERC-8004](https://eips.ethereum.org/EIPS/eip-8004) - エージェントのIdentity、Reputation、Validationレジストリに関するEthereum標準。MetaMask、EF、Google、Coinbaseが共著。
- [MCP](https://modelcontextprotocol.io/) - AnthropicによるAIとツール・データを接続するModel Context Protocol。AAIF（Linux Foundation）管理下。[GitHub](https://github.com/modelcontextprotocol)
- [A2A](https://a2a-protocol.org/) - Googleによるエージェント間通信とタスクライフサイクル管理のためのAgent2Agentプロトコル。[GitHub](https://github.com/a2aproject/A2A)
- [AGENTS.md](https://agents.md/) - AIコーディングエージェントにプロジェクト固有のガイダンスを与えるOpenAIのオープン標準。AAIF管理下。
- [Agentic AI Foundation](https://www.linuxfoundation.org/press/linux-foundation-announces-the-formation-of-the-agentic-ai-foundation) - MCP、goose、AGENTS.mdを統治するLinux Foundation管理ファンド。Platinumメンバー：AWS、Anthropic、Google、Microsoft、OpenAI。
- [Goose](https://block.github.io/goose/) - BlockによるオープンソースAIエージェントフレームワーク。AAIFガバナンス下のMCPリファレンス実装。[GitHub](https://github.com/block/goose)

## 🛠️ エージェントフレームワーク

経済的エージェントを構築するためのインフラ。

- [ElizaOS](https://elizaos.ai/) - Chainlink CCIPによるクロスチェーンサポート付きの暗号資産向け自律エージェントフレームワーク。[GitHub](https://github.com/elizaOS/eliza)
- [CrewAI](https://www.crewai.com/) - Fortune 500企業の60%が使用するマルチエージェントオーケストレーションフレームワーク。[GitHub](https://github.com/crewAIInc/crewAI)
- [Autonolas](https://olas.network/) - コンポーザブルサービスによるオンチェーン自律エージェントサービスプロトコル。[GitHub](https://github.com/valory-xyz)
- [Bittensor](https://bittensor.com/) - 129以上のサブネットで異なるAIタスクを処理する分散型AIネットワーク。[GitHub](https://github.com/opentensor/bittensor)
- [Ocean Protocol](https://oceanprotocol.com/) - プライバシー保護推論のためのCompute-to-Data付き分散型AIデータマーケットプレイス。
